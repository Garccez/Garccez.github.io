This is a simple page made just so people can download a script made by me, where it installs the gothic keyboard layout on linux.

Instructions

Step. 1. Open your favorite terminal.

Step. 2. Turn this script into an executable with chmod +x install_got.sh.

Step. 3. Run the script with sudo ./install_got.sh

Note: If changes don't apply immediately, log off and then log in again.

That's all! Be happy.
